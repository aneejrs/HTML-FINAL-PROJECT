STUDENT PROFILE & FEEDBACK PORTAL

FILES:
1. index.html       –  Student Profile Form
2. dashboard.html   –  Student Dashboard
3. downloads.html   –  Download Center & Media
4.         images   –  images used in forms & gallery
5.         video    –  promo video
6.          docs    –  syllabus pdf

HOW TO USE:
1. Open index.html 
2. Submit the form → redirects to dashboard.html
3. Go to downloads.html from navigation menu
4. Test:
 -   Form inputs
 -  Table
 -  Image, audio, video, iframe
 -   Quotes: q, blockquote, abbr
 -  Pre tag
 -  Download links
 - Internal anchors (#top, #bottom)
 -  target="_blank" and "_self"

NOTES:
- No CSS or JavaScript required.
- All files must remain in the same folder structure.
